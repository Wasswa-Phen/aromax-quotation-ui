# Aromax Quotation UI (Front-end)

A conversion-focused quotation experience for Aromax Technologies and Aromax Productions.
Built with vanilla HTML, CSS, and JavaScript to ensure speed, simplicity, and easy deployment.

## Goals
- Reduce customer confusion and drop-off during quotation requests
- Capture structured requirements with minimal friction
- Provide WhatsApp fallback for fast alignment
- Maintain brand identity (Red / Orange / Black / White), no gradients

## Tech Stack
- HTML5
- CSS3 (component-based styles)
- Vanilla JavaScript (no frameworks)
